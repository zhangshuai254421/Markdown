# 火炬之光无限——伤害计算入门指南

---

## 1. 伤害计算总览——一条公式看懂伤害

### 1.1 核心公式

火炬之光无限中，一次**击中伤害（Hit Damage）** 的计算遵循以下流水线：

```
最终伤害 = 基础伤害
        × (1 + Σ增伤%)
        × (1 + Σ更多伤害A%) × (1 + Σ更多伤害B%) × ...  ← 每个"更多"来源独立相乘
        × 暴击倍率（如果暴击）
        × 抗性减免系数
        × 护甲/减伤系数（物理）
```

### 1.2 伤害流水线（图解）

```
  ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
  │  基础伤害  │ →  │  增加伤害  │ →  │  更多伤害  │ →  │  暴击判定  │ →  │  最终伤害  │
  │  (Flat)   │    │ (Increased)│    │  (More)   │    │  (Crit)   │    │  (Final)  │
  └──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
       │               │               │               │               │
       ▼               ▼               ▼               ▼               ▼
  技能自带伤害    各词缀相加      各来源独立乘     暴击伤害加成    抗性/护甲减免
  + 装备附加点伤  天赋/装备/光环   辅助技能宝石    暴击率判定      后真正造成
  + 光环附加点伤  每种类型分别算   天赋中"更多"     + 基础暴伤      的伤害数值
```

## 2. 基础伤害（Flat Damage）——伤害的"种子"

### 2.1 基础伤害的来源

```
基础伤害 = 技能宝石自带伤害 + 装备附加点伤 + 光环/增益附加点伤 + 其他来源

来源分解：
  ┌─────────────────┐
  │ 技能宝石         │  例：火球术 基础火焰伤害 50~80
  │ (Skill Gem)      │  每个技能有自己的基础伤害数值和效用比
  ├─────────────────┤
  │ 武器附加点伤     │  例：法杖附加 20~35 闪电伤害至法术
  │ (Weapon)         │  注意区分"附加至法术"和"附加至攻击"
  ├─────────────────┤
  │ 戒指/项链/腰带   │  例：戒指词缀 "附加 10~18 火焰伤害至攻击"
  │ (Jewelry)        │
  ├─────────────────┤
  │ 光环/增益        │  例：愤怒光环 "你与周围友军附加 15~25 火焰伤害至法术"
  │ (Aura/Buff)      │
  ├─────────────────┤
  │ 辅助宝石         │  例：附加火焰伤害(辅) "被辅助技能附加 X~Y 火焰伤害"
  │ (Support Gem)    │
  └─────────────────┘
```

### 2.2 技能伤害效用比（Damage Effectiveness）

```
很多技能宝石上会标注"伤害效用比：XX%"

  实际基础伤害 = 技能自带伤害 + (所有附加点伤 × 效用比%)

  例：
    火球术 自带 50 基础火焰伤害，效用比 120%
    装备附加 20 闪电伤害至法术

    实际基础伤害 = 50 + (20 × 120%) = 50 + 24 = 74
```

### 2.3 攻击 vs 法术——基础伤害的区别

```
攻击技能（Attack）：
  基础伤害来源 = 武器基础伤害 × 技能伤害倍率
  附加点伤需标注"附加至攻击"
  受命中率影响

法术技能（Spell）：
  基础伤害来源 = 技能宝石自带伤害
  附加点伤需标注"附加至法术"
  必中，不受命中率影响

  ═══════════════════════════════════
  ║  类型    ║  基础来源  ║  命中判定  ║
  ╠══════════╬═══════════╬═══════════╣
  ║  攻击    ║  武器伤害  ║  需要命中  ║
  ║  法术    ║  宝石数值  ║  必中      ║
  ╚══════════╩═══════════╩═══════════╝
```

## 3. 增伤机制——伤害的"放大镜"

### 3.1 增伤（Increased）——加法叠加

```
增伤% 是所有"提高/增加"类词缀的总和，加法计算：

  总增伤% = 天赋树增伤 + 装备词缀增伤 + 光环增伤 + 其他来源

  例：
    天赋：火焰伤害提高 50%
    天赋：元素伤害提高 30%
    装备：火焰伤害提高 25%
    光环：元素伤害提高 20%

    总增伤 = 50% + 30% + 25% + 20% = 125%

    伤害 = 基础伤害 × (1 + 125%) = 基础伤害 × 2.25
```

### 3.2 更多伤害（More）——乘法叠加

```
"更多伤害"是每个来源独立相乘，这是伤害质变的核心！

  最终伤害 = 基础伤害 × (1 + 增伤%)
                    × (1 + 更多A%)
                    × (1 + 更多B%)
                    × (1 + 更多C%)
                    × ...

  同源的"更多"先加，不同源的"更多"相乘。

  ═══════════════════════════════════════════
  核心区别：
    "提高 50%"  →  加法，和所有其他"提高"相加
    "额外 50%"  →  乘法，独立于所有其他词缀
  ═══════════════════════════════════════════
```

### 3.3 增伤 vs 更多伤害——直观对比

```
假设基础伤害 = 100，已有 200% 增伤

场景 A：再获得 50% 增伤
  伤害 = 100 × (1 + 200% + 50%) = 100 × 3.5 = 350

场景 B：再获得 50% 更多伤害
  伤害 = 100 × (1 + 200%) × (1 + 50%) = 100 × 3 × 1.5 = 450

结论：已有高增伤时，"更多伤害"的收益远超"增伤"！

        1000 ┤                          ╭──── 更多伤害路径
        900  ┤                        ╭─╯
        800  ┤                      ╭─╯
        700  ┤                    ╭─╯
        600  ┤                  ╭─╯
        500  ┤                ╭─╯
        400  ┤              ╭─╯      ╭──── 增伤路径(收益递减)
        300  ┤            ╭─╯      ╭─╯
        200  ┤          ╭─╯      ╭─╯
        100  ┤──────────╯──────╭─╯
           └──┬───┬───┬───┬───┬───┬───┬───
             100 200 300 400 500 600 700
                   已有增伤%
```

### 3.4 常见的"更多伤害"来源

```
辅助宝石（最大来源）：
  - 附加火焰伤害(辅)：更多火焰伤害
  - 元素集中(辅)：更多元素伤害，但无法造成异常
  - 集中效应(辅)：更多范围伤害，但范围缩小
  - 精准破坏(辅)：更多法术伤害，但降低暴击率

天赋树大点：
  - 火焰穿透(大点)：更多火焰伤害

传奇装备特效：
  - 部分传奇装备提供"更多伤害"特效

诅咒/减益：
  - 元素诅咒：使敌人受到的伤害提高（对敌人而言是"增加"）
  - 感电：敌人受到的伤害提高 15%（更多伤害）
```

## 4. 暴击机制——伤害的"倍增器"

### 4.1 暴击判定

```
每次击中独立判定暴击：

  实际暴击率 = 基础暴击率 + 暴击率加成

  命中 → 判定暴击率 → ┬─ 暴击 → 应用暴击伤害倍率
                      └─ 未暴击 → 普通伤害

  例：
    技能基础暴击率 = 6%
    暴击率提高 200%（来自天赋/装备）

    实际暴击率 = 6% × (1 + 200%) = 6% × 3 = 18%
```

### 4.2 暴击伤害

```
暴击时：

  暴击伤害倍率 = 基础暴击伤害倍率 × (1 + 暴击伤害加成%)

  基础暴击伤害倍率 = 150%（即 1.5 倍伤害）
  （部分技能/装备会改变基础暴击伤害倍率）

  例：
    基础暴伤 150%
    暴击伤害加成 100%（天赋/装备）

    暴击伤害倍率 = 150% × (1 + 100%) = 150% × 2 = 300%
    暴击时造成 3 倍伤害

  ┌──────────────────────────────────────┐
  │  非暴击：伤害 = 基础 × (1 + 增伤)    │
  │  暴击：  伤害 = 基础 × (1 + 增伤)    │
  │                × 暴击伤害倍率         │
  └──────────────────────────────────────┘
```

### 4.3 暴击期望伤害

```
暴击期望倍率 = 1 + (暴击率 × (暴击伤害倍率 - 1))

  例：
    暴击率 25%，暴击伤害 300%

    期望倍率 = 1 + (0.25 × (3.0 - 1))
             = 1 + (0.25 × 2)
             = 1 + 0.5 = 1.5

  即平均伤害是不暴击时的 1.5 倍

  ═══════════════════════════════════════════
  ║  暴击率  ║  暴伤  ║  期望倍率  ║
  ╠══════════╬═══════╬═══════════╣
  ║  10%     ║  200%  ║  1.10     ║  前期参考
  ║  25%     ║  250%  ║  1.375    ║  中期参考
  ║  50%     ║  350%  ║  2.25     ║  后期参考
  ║  80%     ║  500%  ║  4.20     ║  毕业参考
  ║  100%    ║  400%  ║  4.00     ║  满暴参考
  ╚══════════╩═══════╩═══════════╝
```

## 5. 抗性与穿透——元素伤害的"破甲"

### 5.1 怪物抗性基础

```
怪物默认拥有元素抗性（火焰/冰霜/闪电/腐蚀）：

  白怪：约 0~20% 抗性
  蓝怪：约 20~35% 抗性
  金怪/首领：约 35~50% 抗性
  终极首领：约 50~75% 抗性（有抗性上限）

  抗性减免 = 1 - 怪物抗性%（下限为负数，即增伤）

  例：怪物火焰抗性 30%
    火焰伤害 = 原始伤害 × (1 - 30%) = 原始伤害 × 0.7
```

### 5.2 穿透（Penetration）

```
穿透降低怪物的实际抗性，直接作用在抗性计算中：

  实际抗性 = 怪物原本抗性 - Σ穿透%

  例：
    怪物火焰抗性 50%
    你有 20% 火焰穿透

    实际抗性 = 50% - 20% = 30%
    伤害倍率 = 1 - 30% = 70%（原先是 50%）

  穿透是直接乘法收益，穿透越高收益越大！

  ┌──────────────────────────────────────────┐
  │  穿透 vs 降抗：                           │
  │  穿透 = 忽视目标抗性（只对自己有效）       │
  │  降抗 = 降低目标抗性（全队受益）           │
  │  两者可以叠加！                           │
  └──────────────────────────────────────────┘
```

### 5.3 抗性穿透收益表

```
  怪物抗性越高，穿透收益越大：

  ═══════════════════════════════════════════════
  ║  怪物抗性  ║  穿透20%  ║  穿透40%  ║  穿透60%  ║
  ╠════════════╬══════════╬══════════╬══════════╣
  ║  0%        ║  1.20×    ║  1.40×    ║  1.60×    ║
  ║  30%       ║  1.28×    ║  1.57×    ║  1.86×    ║  注：倍率 = (1-新抗性)/(1-原抗性)
  ║  50%       ║  1.40×    ║  1.80×    ║  2.20×    ║
  ║  75%       ║  1.80×    ║  2.60×    ║  3.40×    ║
  ╚════════════╩══════════╩══════════╩══════════╝

  关键结论：
  → 高抗性怪物（首领）面前，穿透是最强增伤手段之一！
  → 让怪物抗性变负时，每 1% 穿透 ≈ 1% 更多伤害
```

### 5.4 穿透 vs 更多伤害——如何选择

```
假设你已有 300% 增伤，怪物 50% 抗性，基础伤害 1000：

方案 A：+40% 更多元素伤害
  伤害 = 1000 × (1+300%) × (1+40%) × (1-50%)
       = 1000 × 4 × 1.4 × 0.5 = 2800

方案 B：+40% 元素穿透
  伤害 = 1000 × (1+300%) × (1-10%)
       = 1000 × 4 × 0.9 = 3600

结论：怪物抗性越高，"穿透"优于"更多伤害"！
      怪物抗性越低，"更多伤害"性价比更高。
```

## 6. 伤害转换——练武不练功，到老一场空

### 6.1 转换规则

```
伤害转换是将一种伤害类型变为另一种：

  物理 → 火焰 → 冰霜 → 闪电 → 腐蚀（单向链）

  转换规则：
  1. 只按上述顺序单向转换，不可反向
  2. 同一技能可多重转换，按顺序计算
  3. 转换%超过100%时，按比例缩放
```

### 6.2 转换收益——为什么转换是顶级增伤

```
转换的收益在于：原类型和转换后类型的增伤都生效！

  例：100 物理伤害，50% 转换为火焰伤害

  结果：50 物理伤害 + 50 火焰伤害

  如果你有：
    - 物理伤害提高 100%
    - 火焰伤害提高 100%
    - 元素伤害提高 50%

  最终伤害：
    物理部分 = 50 × (1 + 100%)               = 50 × 2 = 100
    火焰部分 = 50 × (1 + 100% + 50%)         = 50 × 2.5 = 125
    总计     = 100 + 125 = 225

  如果没转换（100 纯物理）：
    = 100 × (1 + 100%) = 200

  转换后：225 > 200（多享受了火焰增伤和元素增伤！）
```

### 6.3 额外伤害（Extra Damage / Gain as）

```
"获得 X% 物理伤害的额外火焰伤害"

这是比转换更强的机制——它不是"改变"，而是"新增"！

  例：100 物理伤害，获得 30% 物理的额外火焰伤害

  结果：100 物理 + 30 火焰 = 130 总基础伤害

  原物理伤害不变，白送火焰伤害！

  ┌──────────────────────────────────────────┐
  │  转换 (Convert)：                         │
  │    100 物 → 50 物 + 50 火 (总量不变)       │
  │                                          │
  │  额外 (Gain as Extra)：                   │
  │    100 物 → 100 物 + 30 火 (总量增加!)     │
  └──────────────────────────────────────────┘
```

### 7.2 异常状态伤害计算

```
点燃伤害（每秒）= 基础火焰伤害 × 点燃伤害倍率 × (1 + 异常伤害加成)
                            × (1 + 持续伤害加成)
                            × (1 + 火焰持续伤害加成)

  关键点：
  - 点燃伤害只看"基础火焰伤害"，暴击不直接加成点燃
  - 但暴击可以触发更强的点燃（通过暴击伤害加成点燃基础）
  - 持续伤害不吃施法速度加成
  - 感电的"受到伤害+15%"对点燃也有效！
```

### 7.3 感电——全伤害放大器

```
感电是最通用的异常，所有伤害类型都能受益：

  感电效果：敌人受到的所有伤害提高 15%
  感电门槛：单次闪电伤害需超过敌人生命值的一定比例
  （门槛由感电几率/效果词缀降低）

  实际应用：
    即使你不是闪电流派，也可以：
    1. 副手带一个闪电技能触发感电
    2. 使用"感电扩散"辅助宝石
    3. 主技能享受 15% 更多伤害加成

  ┌──────────────────────────────────────────┐
  │  感电 + 暴击 + 穿抗 + 降抗 的乘法链：      │
  │  伤害 × 暴伤 × (1-新抗性) × 1.15(感电)    │
  │                                          │
  │  这些全部是乘法关系，互不稀释！            │
  └──────────────────────────────────────────┘
```

## 8. 持续伤害（DoT）——时间是你的武器

### 8.1 持续伤害类型

```
直接持续伤害（技能本身是 DoT）：
  - 瘟疫（腐蚀 DoT）
  - 灼热射线（火焰 DoT）
  - 旋涡（冰霜 DoT）

异常持续伤害（由击中触发）：
  - 点燃（火焰暴击/几率触发）
  - 中毒（物理+混沌/腐蚀几率触发）
  - 流血（物理攻击几率触发）
```

### 8.2 DoT 伤害计算

```
DoT 每秒伤害 = 基础伤害 × (1 + 增伤%)
                     × (1 + 更多DoT伤害%)
                     × (1 + 更多对应类型伤害%)
                     × (1 + 持续伤害加成%)

  DoT 不受以下属性影响：
    ✗ 施法/攻击速度
    ✗ 暴击（直接部分）
    ✗ 命中率
    ✗ 穿透（部分，取决于具体 DoT 机制）

  DoT 可以享受：
    ✓ 增伤 / 更多伤害
    ✓ 持续伤害加成（专用词缀）
    ✓ 感电（目标受到伤害提高）
    ✓ 降抗（目标抗性降低）
```

### 8.3 DoT 与 Hit 的取舍

```
DoT DPS = DoT 每跳伤害 × 叠加层数 × 持续时间（秒）

  优势：
    - 忽略攻速/施法速度，装备压力小
    - 放一个技能可以跑位，输出不中断
    - 无视护甲减伤（腐蚀/混沌 DoT）

  劣势：
    - 不吃暴击，爆发力不如 Hit 流
    - 伤害有延迟，不能秒怪
    - 对高回复敌人效果打折

  ═══════════════════════════════════════════
  ║  场景      ║  推荐流派               ║
  ╠════════════╬════════════════════════╣
  ║  速刷清图  ║  Hit 流（秒怪）         ║
  ║  首领战    ║  DoT 流（边跑边输出）   ║
  ║  攻坚      ║  Hit 暴击流（爆发秒阶段）║
  ║  HC模式    ║  DoT 流（安全）         ║
  ╚════════════╩════════════════════════╝
```

## 9. 防御属性如何影响你的伤害

### 9.1 怪物护甲——物理伤害的克星

```
物理伤害受目标护甲减免：

  护甲减伤% = 护甲值 / (护甲值 + 伤害值 × K)

  （K 为常数，不同游戏版本可能不同）

  关键结论：
  1. 单次伤害越大，护甲减免效果越差（护甲怕大伤害）
  2. 小伤害高频次 → 护甲减免多
  3. 大伤害低频次 → 护甲减免少

  应对策略：
    - 物理穿透 / 粉碎护甲（降低目标护甲值）
    - 伤害转换（把物理转成元素，绕开护甲）
    - 堆高单次伤害（利用护甲对大伤害减免差的特性）
```

### 9.2 躲避、闪避与格挡

```
攻击技能需要过三层"miss"判定：

  1. 命中率判定：
     实际命中率 = 你的命中值 / (命中值 + 目标闪避值^0.8)
     命中不足 → 大量攻击打空，DPS 严重受损

  2. 躲避判定（攻击专属）：
     即使命中，目标也有几率完全躲避

  3. 格挡判定：
     即使没躲避，目标也有几率格挡（减少或免疫伤害）

  法术必中，但可以被"法术格挡"和"法术压制"减免

  ┌──────────────────────────────────────────────┐
  │  攻击流派必须堆命中！否则面板 DPS 是虚的。     │
  │  法术流派不需要命中，专注伤害面即可。          │
  └──────────────────────────────────────────────┘
```

## 10. 实战计算案例——手把手算伤害

### 10.1 案例设定

```
角色配置：
  - 技能：火球术（8 级）
    基础火焰伤害：120~180（取平均 150）
    伤害效用比：100%
    基础暴击率：6%

  - 武器：法杖，附加 25~45 闪电伤害至法术（平均 35）
  - 戒指：附加 12~20 火焰伤害至法术（平均 16）
  - 天赋：火焰伤害提高 80%，元素伤害提高 40%，法术伤害提高 30%
  - 辅助宝石：
    - 附加火焰伤害(辅)：更多火焰伤害 35%
    - 元素集中(辅)：更多元素伤害 30%
  - 暴击属性：暴击率提高 150%，暴击伤害加成 80%
  - 火焰穿透：25%

目标怪物：火焰抗性 45%
```

### 10.2 逐步计算

```
第一步：基础伤害
  = 技能基础(150) + 法杖附加(35) + 戒指附加(16)
  = 150 + 35 + 16
  = 201（平均基础伤害）

第二步：总增伤
  = 1 + (80% + 40% + 30%)
  = 1 + 150%
  = 2.50

第三步：更多伤害
  = (1 + 35%) × (1 + 30%)
  = 1.35 × 1.30
  = 1.755

第四步：暴击期望
  实际暴击率 = 6% × (1 + 150%) = 6% × 2.5 = 15%
  暴击伤害倍率 = 150% × (1 + 80%) = 150% × 1.8 = 270%
  暴击期望倍率 = 1 + (15% × (270% - 1)) = 1 + (0.15 × 1.7)
               = 1 + 0.255 = 1.255

  非暴击伤害线（用于参考）：
    暴击时 = 按 2.70 倍
    未暴击时 = 按 1.00 倍

第五步：抗性减免
  实际火焰抗性 = 45% - 25% = 20%
  抗性系数 = 1 - 20% = 0.80

第六步：最终期望伤害（单次击中）
  期望伤害 = 201 × 2.50 × 1.755 × 1.255 × 0.80
           = 201 × 2.50 × 1.755 × 1.255 × 0.80
           = 201 × 4.403... × 0.80
           ≈ 885（期望值）

  暴击单次伤害：
  暴击伤害 = 201 × 2.50 × 1.755 × 2.70 × 0.80
           ≈ 201 × 9.477 × 0.80
           ≈ 1524

  非暴击单次伤害：
  非暴击伤害 = 201 × 2.50 × 1.755 × 1.00 × 0.80
             ≈ 201 × 4.388 × 0.80
             ≈ 705
```

### 10.3 伤害提升优先级分析

```
从 10.2 案例出发，对比各项提升效果：

当前倍率结构：
  增伤 × 更多 × 暴击期望 × 抗性
  = 2.50 × 1.755 × 1.255 × 0.80
  = 4.40

各选项提升效果：
  - 加 50% 增伤      → 3.00 × 1.755 × 1.255 × 0.80 = 5.29 → +20.1%
  - 加 30% 更多伤害   → 2.50 × 2.282 × 1.255 × 0.80 = 5.73 → +30.0%
  - 暴击率 +15%       → 2.50 × 1.755 × 1.425 × 0.80 = 5.00 → +13.5%
  - 穿透 +20%         → 2.50 × 1.755 × 1.255 × 1.00 = 5.50 → +25.0%

优先级结论：
  穿透(25%) > 更多伤害(30%) > 增伤(20.1%) > 暴击(13.5%)

  注意：这是基于当前案例数据的排序，实际游戏中的最优选择
  取决于你已有的各项属性——每项都存在"边际效益递减"！
```

## 11. 伤害提升的底层逻辑——边际效益递减

### 11.1 什么是边际效益递减

```
每项属性的收益不是线性的——堆得越多，每 1% 的收益越低：

  ┌──────────────────────────────────────────────┐
  │  增伤：                                      │
  │    0→100% 增伤：伤害 ×2（+100%）              │
  │    300→400% 增伤：伤害从 4×→5×（仅+25%）      │
  │                                              │
  │  同样是 +100% 增伤，实际收益从 100% 降到 25%！ │
  └──────────────────────────────────────────────┘
```

### 11.2 配装思路——全面发展

```
        伤害最大化 = 各项乘区均衡发展

  ═══════════════════════════════════
          基础伤害
            │
      ┌─────┼─────┐
      ▼     ▼     ▼
    增伤  更多伤害  暴击
      │     │     │
      └─────┼─────┘
            ▼
         穿透
            │
            ▼
        最终伤害
  ═══════════════════════════════════

  核心原则：
    1. 优先补齐短板（收益最大的乘区）
    2. 同类加法叠加的词缀适度即可，不要无脑堆
    3. 不同乘区的词缀优先获取（乘法关系）
    4. 高抗性敌人 → 穿透优先；低抗性敌人 → 更多伤害优先
    5. 暴击投资需要暴击率和暴伤两手抓，缺一不可
```

## 12. 面板伤害 vs 实际伤害——别被面板骗了

```
游戏内面板显示的 DPS 不会计算以下因素：

  面板不计算（但实际生效）：
    ✓ 穿透
    ✓ 降抗
    ✓ 感电增伤
    ✓ 诅咒/减益
    ✓ 敌人护甲减免
    ✓ 命中率（攻击流派）
    ✓ 多阶段伤害（如投射物返回）

  面板计算了（但可能虚高）：
    ✗ 假设 100% 命中（攻击流派）
    ✗ 假设 0% 抗性
    ✗ 不计怪物减伤

  结论：面板 DPS 只是参考！
        两个同样 10W 面板的角色，实际伤害可能差 2~3 倍。
```
