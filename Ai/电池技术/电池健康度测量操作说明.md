## 1. 什么是电池健康度

电池健康度（Battery Health / State of Health, **SOH**）是用百分比表示的电池当前可用容量相对于原始设计容量的比值。

```
SOH(%) = 当前最大容量 ÷ 出厂设计容量 × 100%
```

| 健康度 | 状态 | 建议 |
|--------|------|------|
| 90% ~ 100% | 优秀 | 正常使用 |
| 80% ~ 90% | 良好 | 正常使用 |
| 70% ~ 80% | 一般 | 开始关注，减少深度放电 |
| 60% ~ 70% | 较差 | 考虑更换 |
| < 60% | 严重老化 | 建议更换 |

## 2. Windows —— 系统内置方法

### 方法 1：powercfg 电池报告（推荐，最详细）

```powershell
# 以管理员身份打开 PowerShell，执行：
powercfg /batteryreport /output "C:\Users\%USERNAME%\Desktop\battery-report.html"
```

打开桌面生成的 `battery-report.html`，关键指标：

| 指标 | 位置 | 含义 |
|------|------|------|
| **DESIGN CAPACITY** | Installed batteries 表格 | 出厂设计容量（mWh） |
| **FULL CHARGE CAPACITY** | Installed batteries 表格 | 当前充满时的最大容量（mWh） |
| **CYCLE COUNT** | Installed batteries 表格 | 已完成的充放电循环次数 |

**计算健康度**：
```
健康度 = FULL CHARGE CAPACITY ÷ DESIGN CAPACITY × 100%
```

例如：`48,000 mWh ÷ 56,000 mWh × 100% = 85.7%`

### 方法 2：powercfg 能量估算

```powershell
# 生成能量估算报告（包含耗电分析）
powercfg /energy /output "C:\Users\%USERNAME%\Desktop\energy-report.html" /duration 60
```

### 方法 3：WMIC 快速查询

```cmd
wmic path Win32_Battery Get EstimatedChargeRemaining
```

> ⚠️ 注意：仅返回当前电量百分比，非健康度。

## 3. macOS —— 系统内置方法

### 方法 1：系统信息（推荐）

```
1. 按住 ⌥ Option 键，点击菜单栏右上角电池图标
2. 查看「状况」字段：
   - 正常（Normal）    → 健康
   - 建议维修（Service Recommended） → 健康度 < 80%
```

### 方法 2：系统报告

```
1. 苹果菜单 → 关于本机 → 系统报告
2. 左侧选择「电源」
3. 查看关键指标：
```

| 指标 | 含义 |
|------|------|
| **循环计数** | 已完成的充放电循环次数 |
| **最大容量** | 当前电池最大容量（mAh） |
| **状态** | 正常 / 建议维修 / 更换 |

### 方法 3：终端命令

```bash
# 查看电池信息
system_profiler SPPowerDataType

# 查看循环次数
pmset -g batt

# 查看原始数据（含健康百分比）
ioreg -r -c "AppleSmartBattery" | grep -i capacity
```

输出示例：
```
"AppleRawMaxCapacity" = 4100     # 当前最大容量
"DesignCapacity"       = 4800    # 出厂设计容量
# 健康度 = 4100 ÷ 4800 = 85.4%
```

## 4. Linux —— 命令行方法

### 读取电池信息

```bash
# 查看电池信息
upower -i /org/freedesktop/UPower/devices/battery_BAT0
```

关键输出：
```
energy-full:         38.24 Wh        ← 当前最大容量
energy-full-design:  48.00 Wh        ← 出厂设计容量
capacity:            79.7%           ← 健康度（直接给出）
```

### 或直接从 sysfs 读取

```bash
cat /sys/class/power_supply/BAT0/capacity          # 当前电量%
cat /sys/class/power_supply/BAT0/energy_full       # 当前最大容量
cat /sys/class/power_supply/BAT0/energy_full_design # 出厂设计容量
cat /sys/class/power_supply/BAT0/cycle_count        # 循环次数
```

## 5. Android —— 应用与命令

### 方法 1：ADB 命令（无需 root）

```bash
# 查看电池信息
adb shell dumpsys battery
```

输出示例：
```
level: 85                    # 当前电量%
health: 2                    # 2 = Good
```

### 方法 2：ADB 健康度（需调试模式）

```bash
adb shell dumpsys batterymanager
```

### 方法 3：第三方应用

| 应用 | 说明 |
|------|------|
| **AccuBattery** | 充电循环中测量实际充入电量，持续使用 1~2 周得出估计健康度 |
| **DevCheck** | 展示硬件信息，含电池健康度（需内核支持） |
| **Ampere** | 测量充放电电流，间接判断电池状态 |

## 6. iOS —— 系统设置

### 方法 1：内置电池健康

```
设置 → 电池 → 电池健康与充电 → 最大容量
```

| 显示 | 含义 |
|------|------|
| 最大容量：95% | 健康度为 95% |
| 峰值性能容量 | 显示电池是否支持峰值性能 |

### 方法 2：隐私分析数据（详细）

```
设置 → 隐私 → 分析与改进 → 分析数据
→ 搜索 "log-aggregated" → 查找最新文件
→ 搜索 "battery.cycle_count" 和 "battery.design_capacity"
```

```bash
# 使用 ideviceinfo（来自 libimobiledevice）
ideviceinfo -q com.apple.mobile.battery | grep -E "BatteryCurrentCapacity|BatteryIsCharging|DesignCapacity"
```

## 7. 使用万用表测量

适用范围：可拆卸电池（18650、AA/AAA、软包电池等）。

### 7.1 所需工具

- 数字万用表
- 电池

### 7.2 测量步骤

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 万用表拨到 **直流电压档**（DCV） | 选择量程 > 电池标称电压 |
| 2 | 红表笔接 + 极，黑表笔接 - 极 | 注意极性 |
| 3 | 读取电压值 | 与标称电压对比 |
| 4 | 给电池加载额定负载 | 仅电压不够，需测带载压降 |
| 5 | 测量负载下电压 | 如果电压迅速下降 → 电池老化 |

### 7.3 电压与健康度对照（3.7V 锂电池）

| 电压（空载） | 电压（负载） | 状态 |
|-------------|------------|------|
| 4.15V ~ 4.20V | ≥ 3.8V | 满电，健康 |
| 3.80V ~ 4.10V | ≥ 3.6V | 部分放电，正常 |
| 3.60V ~ 3.80V | ≥ 3.4V | 正常放电区间 |
| 3.00V ~ 3.60V | < 3.2V | 需要充电 |
| < 3.00V | — | 过放，可能永久损坏 |
| 充满后（4.2V）放置 24h 跌至 < 4.0V | — | 自放电严重，老化 |

> ⚠️ 电压**不能直接得出健康度**。只能作为快速判断的依据。完整测量需要放电法（见第 8 节）。

## 8. 使用专业电池测试仪

### 8.1 设备推荐

| 设备类型 | 价格区间 | 功能 |
|----------|:------:|------|
| **恒流电子负载**（如 DL24） | ￥200~500 | 设定放电电流，自动记录容量 |
| **锂电池内阻测试仪** | ￥100~300 | 内阻是判断健康的直接指标 |
| **综合充电测试仪**（如 Lii-500） | ￥150~300 | 一次测试容量、内阻 |
| **BMS 采集仪** | ￥500+ | 适用于电池包整体测试 |

### 8.2 测试步骤

```
1. 充满电池（到厂家指定的截止电压）
2. 静置 30 分钟
3. 用电子负载以标准倍率放电（例如 1C→ 2A）
4. 记录放电至截止电压时累计的容量（mAh / mWh）
5. 健康度 = 实测容量 ÷ 标称容量 × 100%
```

### 8.3 内阻法判断（更快）

内阻（mΩ）越高 → 电池越老化：

| 电池类型 | 新的内阻 | 严重老化的内阻 |
|----------|:-------:|:-------------:|
| 18650 锂电 | 20 ~ 50 mΩ | > 80 mΩ |
| 手机电池 | 50 ~ 100 mΩ | > 200 mΩ |
| 汽车启动电池 | 3 ~ 10 mΩ | > 15 mΩ |
| 铅酸电池 | 5 ~ 20 mΩ | > 30 mΩ |

## 9. 电池健康度计算公式

### 基本公式

```
SOH_capacity = C_current / C_rated × 100%

其中：
  C_current = 当前电池从充满到截止电压能放出的容量
  C_rated   = 新电池的标称容量
```

### 内阻公式

```
SOH_IR = (R_endOfLife - R_current) / (R_endOfLife - R_new) × 100%

其中：
  R_new        = 新电池典型内阻
  R_current    = 当前测得内阻
  R_endOfLife  = 报废时的内阻（通常 = 2 × R_new）
```

### 循环次数公式

```
SOH_cycle = (N_max - N_current) / N_max × 100%

其中：
  N_current = 当前循环次数
  N_max     = 电池标称循环寿命（如 锂电通常 300~500 次）
```

### 综合公式（多因素）

```
SOH = w1 × SOH_capacity + w2 × SOH_IR + w3 × SOH_cycle

其中 w1 + w2 + w3 = 1.0（权重按实际调整）
```

## 10. 常见电池类型与判断标准

### 锂离子电池（Li-ion）

| 指标 | 正常范围 | 警告 | 严重 |
|------|:------:|:----:|:----:|
| SOH | > 80% | 60~80% | < 60% |
| 内阻增量 | < 20% | 20~50% | > 50% |
| 充满电压 | 4.18~4.22V | 4.15~4.18V | < 4.15V |
| 循环次数 | < 300 | 300~500 | > 500 |
| 自放电率/月 | < 2% | 2~5% | > 5% |

### 磷酸铁锂电池（LiFePO₄）

| 指标 | 正常范围 | 警告 |
|------|:------:|:----:|
| SOH | > 85% | < 85% |
| 充满电压 | 3.55~3.65V | < 3.50V |
| 内阻 | < 10 mΩ | > 15 mΩ |
| 循环次数 | < 2000 | > 2000 |

### 铅酸电池

| 指标 | 正常 | 警告 | 严重 |
|------|:---:|:----:|:----:|
| 静止电压（6 单体） | > 12.6V | 12.2~12.6V | < 12.2V |
| 负载 10s 恢复时间 | < 5s | 5~15s | > 15s |
| 内阻 | < 10 mΩ | 10~30 mΩ | > 30 mΩ |
