## 1. 总览：判断一块二手电池好坏的 5 个维度

```
                 ┌──────────────┐
                 │  外观检查     │ ← 第一步：有无鼓包、漏液、锈蚀
                 └──────┬───────┘
                        ↓
                 ┌──────────────┐
                 │  电压测量     │ ← 第二步：是否在正常电压区间
                 └──────┬───────┘
                        ↓
                 ┌──────────────┐
                 │  内阻测量     │ ← 第三步：最关键指标，决定电池还能不能用
                 └──────┬───────┘
                        ↓
                 ┌──────────────┐
                 │  容量实测     │ ← 第四步：以实际放电数据说话
                 └──────┬───────┘
                        ↓
                 ┌──────────────┐
                 │  自放电测试   │ ← 第五步：充满后静置，看电压跌落
                 └──────────────┘
```

> **核心原则**：二手电池只有一个真理——**实测容量 ÷ 标称容量 = 真正的健康度**。外观、电压、内阻都是辅助判断，最终要上放电仪。

## 2. 第一步：外观检查

这是最快、零成本的筛选，直接排除硬伤的电池。

### 2.1 逐项检查清单

| 检查项 | 正常 | 不合格（退货） |
|--------|------|--------------|
| **鼓包** | 表面平整，无凸起 | 任何肉眼可见的鼓包 → 内部产气，**直接退货** |
| **漏液** | 极柱/泄压阀周围干燥 | 有白色结晶、湿痕 → 电解液泄漏 |
| **锈蚀** | 极柱光亮或正常氧化色 | 绿色/白色粉末堆积、极柱锈穿 |
| **凹陷/变形** | 壳体平整 | 深度 > 1mm 的凹坑、铝壳明显挤压 |
| **破损** | 外壳完整 | 裂纹、穿孔 → 内部短路风险 |
| **泄压阀** | 未开启、无异味 | 阀口有喷出物痕迹 → 已触发过安全阀 |
| **生产日期** | 见壳体喷码 | 超过 3 年的新电池也不建议；二手已经用了超过标称循环寿命的不买 |
| **极柱平整度** | 螺丝孔完好、无滑丝 | 滑丝、焊点脱落 |

### 2.2 气味判断

```
正常的锂电池：几乎无味，或极轻微的化学气味
有问题的电池：
  - 甜味/果味 → 电解液泄漏（LiPF₆ 分解）
  - 刺激性酸味 → 过充/过放后的产物
  - 有明显异味 → 直接退货
```

## 3. 第二步：电压测量

### 3.1 工具

- 万用表（精度 ≥ 0.01V）
- 或者带电压显示的电池测试仪

### 3.2 各类型电池的正常电压区间

| 电池类型 | 标称电压 | 满电电压 | 放到宝的截止电压 | **收到货应有的电压** | 不合格 |
|----------|:------:|:------:|:------:|:------:|------|
| **磷酸铁锂（单体）** | 3.2V | 3.60~3.65V | 2.50V | **3.20V ~ 3.35V** | < 2.50V 或 > 3.65V |
| **三元锂（单体）** | 3.7V | 4.20V | 2.75V | **3.60V ~ 3.85V** | < 2.75V 或 > 4.25V |
| **刀片电池（LFP）** | 3.2V | 3.65V | 2.50V | **3.20V ~ 3.35V** | < 2.50V |
| **钛酸锂（单体）** | 2.4V | 2.85V | 1.80V | **2.40V ~ 2.60V** | < 1.80V |
| **铅酸（12V）** | 12.0V | 12.8V | 10.5V | **12.5V ~ 12.8V** | < 10.5V |
| **铅酸（单体 2V）** | 2.0V | 2.13V | 1.75V | **2.08V ~ 2.13V** | < 1.75V |

### 3.3 电压能告诉你的信息

```
收到货后测量电压：

电压在「收到货应有电压」区间内
  → ✅ 正常，电池存放状态良好，继续测下一步

电压低于截止电压（如磷酸铁锂 < 2.5V）
  → ⚠️ 过放，可能已造成永久容量损伤
  → 可以尝试用 0.1C 小电流充电到 3.0V，再做容量测试
  → 但这种情况不要抱太大期望

电压为 0V
  → ❌ 内部断路或严重短路，直接退货

电压高于满电电压
  → ❌ 卖家用非常规手段充过电，可能已过充损伤

同一批多块电池，电压差异 > 0.05V
  → ⚠️ 可能是不同批次的拆机电池混在一起卖
```

### 3.4 操作要点

```
1. 万用表打到 DCV 档（直流电压）
2. 量程选 20V（单体）/ 200V（整组）
3. 红笔接正极（+），黑笔接负极（−）
4. 记录每块电池的电压值
5. 同一批买的，所有电池电压应该非常接近（±0.02V以内最优）
```

## 4. 第三步：内阻测量——最关键的指标

**内阻是判断二手电池能不能用的最关键指标**。内阻升高 = 电池老化 = 带载能力下降 = 大电流放电时发热严重。

### 4.1 为什么要测内阻而不是只看电压

```
一个真实的二手电池场景：

电池 A：电压 3.30V，内阻 2.5mΩ → ✅ 好电池
电池 B：电压 3.30V，内阻 8.0mΩ → ❌ 已经严重老化

如果不测内阻，只看电压，A 和 B 的电压一模一样！
装上设备一跑大电流：
  - 电池 A：压降正常，稳定输出
  - 电池 B：电压瞬间掉到截止电压以下，设备断电/保护
```

### 4.2 用什么工具测

| 工具 | 价格 | 精度 | 推荐程度 |
|------|:----:|:----:|:----:|
| **YR1035+** 内阻测试仪 | ￥150~200 | ±0.5% | ⭐⭐⭐⭐⭐ 性价比首选 |
| **RC3563** 内阻测试仪 | ￥200~300 | ±0.3% | ⭐⭐⭐⭐⭐ 精度更高 |
| **TS457** 内阻测试仪 | ￥80~120 | ±0.5% | ⭐⭐⭐⭐ 入门够用 |
| **4 线毫欧表** | ￥500+ | ±0.1% | ⭐⭐⭐⭐⭐ 专业级 |
| 普通万用表 | — | 测不了 | ❌ 万用表测不了毫欧级内阻 |

> ⚠️ **普通万用表的电阻档无法准确测量毫欧级内阻**。必须用**交流 1kHz 四线法**的电池内阻测试仪。这是最值得花的一笔钱。

### 4.3 各类型电池内阻判断标准

#### 磷酸铁锂单体（标称容量对应的新电池内阻）

| 容量 | 新电池内阻 | 良好（二手可接受） | 老化 | 报废 |
|------|:------:|:------:|:----:|:----:|
| 25Ah | ≤ 0.8 mΩ | ≤ 1.0 mΩ | 1.0~1.5 mΩ | > 1.5 mΩ |
| 50Ah | ≤ 0.5 mΩ | ≤ 0.7 mΩ | 0.7~1.0 mΩ | > 1.0 mΩ |
| 100Ah | ≤ 0.3 mΩ | ≤ 0.5 mΩ | 0.5~0.8 mΩ | > 0.8 mΩ |
| 200Ah | ≤ 0.2 mΩ | ≤ 0.3 mΩ | 0.3~0.5 mΩ | > 0.5 mΩ |
| 280Ah（如 EVE） | ≤ 0.15 mΩ | ≤ 0.25 mΩ | 0.25~0.4 mΩ | > 0.4 mΩ |

**经验公式**（磷酸铁锂）：
```
内阻上限 ≈ 30 / 标称容量(Ah)   → 超过此值 = 老化严重
例如 100Ah 电池：30/100 = 0.3mΩ 为上限
```

#### 刀片电池（LFP）

| 型号 | 典型新内阻 | 二手可接受 | 不合格 |
|------|:------:|:------:|:----:|
| 比亚迪刀片（138Ah） | 0.15~0.25 mΩ | ≤ 0.35 mΩ | > 0.4 mΩ |
| 比亚迪刀片（150Ah） | 0.12~0.20 mΩ | ≤ 0.30 mΩ | > 0.35 mΩ |
| 比亚迪刀片（200Ah） | 0.10~0.18 mΩ | ≤ 0.25 mΩ | > 0.30 mΩ |

#### 大单体/铝壳（三元锂）

| 容量 | 新电池内阻 | 二手可接受 | 报废 |
|------|:------:|:------:|:----:|
| 37Ah | ≤ 1.0 mΩ | ≤ 1.5 mΩ | > 2.0 mΩ |
| 50Ah | ≤ 0.8 mΩ | ≤ 1.2 mΩ | > 1.8 mΩ |
| 94Ah | ≤ 0.5 mΩ | ≤ 0.8 mΩ | > 1.2 mΩ |
| 117Ah | ≤ 0.4 mΩ | ≤ 0.6 mΩ | > 1.0 mΩ |

#### 18650 / 21700 圆柱电池

| 类型 | 新电池内阻 | 二手可接受 | 报废 |
|------|:------:|:------:|:----:|
| 动力型 18650（如 VTC6） | 12~18 mΩ | ≤ 25 mΩ | > 30 mΩ |
| 容量型 18650（如 NCR18650GA） | 30~40 mΩ | ≤ 55 mΩ | > 70 mΩ |
| 动力型 21700（如 40T） | 8~12 mΩ | ≤ 18 mΩ | > 25 mΩ |
| 容量型 21700（如 50E） | 15~25 mΩ | ≤ 35 mΩ | > 45 mΩ |

### 4.4 内阻一致性判断（同一批电池）

```
买回来一组（比如 16 块）电池：

✅ 好电池：
  - 所有电池内阻波动在 ±15% 以内
  - 例如 100Ah 电池：0.35 / 0.33 / 0.34 / 0.36 / 0.32 → 一致性极好

⚠️ 可疑：
  - 个别电池内阻明显偏高（高于全组平均值 30% 以上）
  - 例如：0.33 / 0.34 / 0.35 / 0.55 / 0.32 → 这块 0.55 是次品混好卖

❌ 直接退货：
  - 内阻差异 > 50%，说明是拆机电池拼凑的杂货
```

### 4.5 测量操作要点

```
1. 四线测试夹：红色一对夹正极，黑色一对夹负极
2. 夹的位置要靠近极柱根部（不要夹在螺栓或连线上）
3. 等待读数稳定（约 2~3 秒）
4. 记录每块电池的内阻值
5. 同一块电池正反向各测一次，取平均值

⚠️ 刚充电或放电完的电池内阻会偏低（温度高），
   建议在室温下静置 2 小时后再测
```

## 5. 第四步：容量实测——放电法

**这是唯一的真标准**。所有其他测量都是辅助判断，最终以放电仪测出的容量为准。

### 5.1 所需设备

| 设备 | 推荐型号 | 用途 |
|------|---------|------|
| **可调恒流电子负载** | DL24/P / DLB-600 | 设定放电电流，自动截止 |
| **容量测试仪** | EBC-A40L / ZKETECH EBC | 支持充放电，自动出曲线 |
| **库仑计** | 胜利 DM-50C / TK15 | 串联在放电回路中计量 Wh/Ah |
| **可调充电器** | 根据电池电压选型 | 先充满再放电 |

### 5.2 测试步骤

```
┌──────────────────────────────────────────────────┐
│ 第 1 步：充满                                       │
│ 用 CC/CV 充电器充到截止电压                          │
│ 磷酸铁锂：3.65V，电流降到 0.05C 时判定充满            │
│ 三元锂：  4.20V，电流降到 0.05C 时判定充满            │
├──────────────────────────────────────────────────┤
│ 第 2 步：静置 30~60 分钟                             │
│ 让电池电压稳定到标称值附近                            │
├──────────────────────────────────────────────────┤
│ 第 3 步：恒流放电                                    │
│ 放电倍率：0.2C ~ 0.5C（不要太大，测的是容量不是功率）    │
│ 例如 100Ah 电池，用 20A 放电（0.2C）                  │
│                                                     │
│ 截止电压：                                           │
│   磷酸铁锂：2.50V                                    │
│   三元锂：2.75V                                      │
│   刀片电池：同磷酸铁锂，2.50V                          │
├──────────────────────────────────────────────────┤
│ 第 4 步：记录放电容量                                 │
│ 电子负载/库仑计显示的 Ah 或 Wh 值                     │
│                                                     │
│ 健康度 = 实测容量 ÷ 标称容量 × 100%                    │
└──────────────────────────────────────────────────┘
```

### 5.3 容量判定标准

| 实际容量百分比 | 等级 | 二手是否可接受 |
|:------------:|:----:|:------------:|
| ≥ 95% | 优秀 | ✅ 跟新的差不多 |
| 90% ~ 94% | 良好 | ✅ 完全可接受 |
| 85% ~ 89% | 正常 | ✅ 二手正常水平 |
| 80% ~ 85% | 可用 | ⚠️ 价格够便宜可以接受，用于小电流场景 |
| 70% ~ 79% | 老化 | ⚠️ 只能用于对容量要求不高的场景（如应急电源） |
| < 70% | 报废级 | ❌ 退货运费都别省 |

### 5.4 放电曲线的大致特征

```
                   电压
                    ↑
   充满电压 → 3.65V ┤╲
                    │ ╲
                    │  ╲___________ ← 平台区（主要工作区间）
                    │              ╲
    截止电压 → 2.50V ┤               ╲
                    └──────────────────→ 时间

    好电池的特征：
      - 平台区又长又平（占放电时间的 80% 以上）
      - 末端跌落陡峭（说明内阻低）
      - 放出的容量接近标称值

    老化电池的特征：
      - 平台区缩短（容量缩小）
      - 放电一开始电压就偏低（内阻大，压降大）
      - 提前到达截止电压
```

## 6. 第五步：自放电率测试

二手电池另一个核心风险：**看着有容量，放几天就没电了**。

### 6.1 测试方法

```
1. 将电池充满（到厂家截止电压）
2. 静置 30 分钟，记录电压 V₀
3. 室温存放 72 小时（3 天），期间不接任何负载
4. 测量电压 V₁
5. 计算压降：ΔV = V₀ − V₁
```

### 6.2 自放电判断标准

| 电池类型 | 正常（72h 压降） | 可疑 | 严重（退货） |
|----------|:------:|:----:|:----------:|
| **磷酸铁锂（新/好）** | < 0.02V | 0.02~0.05V | > 0.05V |
| **磷酸铁锂（二手可用）** | < 0.03V | 0.03~0.08V | > 0.08V |
| **三元锂** | < 0.03V | 0.03~0.08V | > 0.08V |
| **刀片电池** | < 0.02V | 0.02~0.05V | > 0.05V |
| **铅酸（12V）** | < 0.1V/月 | — | > 0.5V/月 |

### 6.3 快速自放电筛选（简化版）

```
如果等不了 72 小时，可以这样做：

1. 充满电池，记录电压
2. 静置 24 小时
3. 再测电压

24 小时内压降 > 0.03V → 自放电偏大，建议退
```

## 7. 磷酸铁锂（LiFePO₄）专项

磷酸铁锂是目前二手市场流通量最大的类型，专门讲讲。

### 7.1 常见二手来源

| 来源 | 典型状态 | 注意点 |
|------|---------|--------|
| **储能站退役** | 容量 80~90%，循环 200~300 次 | 内阻通常还不错 |
| **电动汽车拆机** | 容量 70~90%，差异大 | 同一车拆的，一致性较好 |
| **通信基站退服** | 容量 85~95%，浮充使用 | 循环少，但长期浮充可能有内部衰减 |
| **测试/库存新电池** | 容量 95~100%，但可能存放时间长 | 主要检查自放电 |
| **事故车拆机** | 不确定 | 可能有隐性损伤，**不建议购买** |

### 7.2 磷酸铁锂特殊注意事项

```
1. 电压平台非常平 → 3.25V~3.35V 区间电压几乎不变
   → 电压不能判断容量！必须放电实测
   → 5 块电池可能都是 3.30V，但容量可能从 60% 到 95% 都有

2. 低温性能差 → 0°C 以下容量大幅缩水
   → 冬天买的电池，在室温下测试

3. 鼓包 = 不可逆损伤 → 磷酸铁锂不像三元那么容易鼓包
   → 一旦鼓包，说明已经严重过充/过放

4. 长期放置在半电最健康 → 卖家如果发来时电压在 3.25V~3.30V
   → 说明对方懂行，存放得当
```

### 7.3 知名品牌参考内阻值

| **EVE 亿纬** | LF304 | 304Ah | 0.10~0.15 mΩ | ≤ 0.22 mΩ |
| **CATL 宁德** | LFP271 | 271Ah | 0.12~0.18 mΩ | ≤ 0.25 mΩ |
| **BYD 弗迪** | LFP100 | 100Ah | 0.20~0.30 mΩ | ≤ 0.45 mΩ |
| **CALB 中航** | L173F163 | 163Ah | 0.15~0.22 mΩ | ≤ 0.35 mΩ |
| **国轩** | IFP27175200 | 200Ah | 0.18~0.25 mΩ | ≤ 0.40 mΩ |

## 8. 刀片电池专项

比亚迪刀片电池（Blade Battery）结构特殊，有单独的检测要点。

### 8.1 刀片电池的特殊结构

```
刀片电池 = 超长超薄的磷酸铁锂单体

典型尺寸（比亚迪）：960mm × 90mm × 13.5mm

结构特点：
  - 壳体自身承重（无模组）
  - 极柱在两端（一端正极一端负极）
  - 外壳就是散热面
```

### 8.2 刀片电池附加检查项

| 检查项 | 方法 | 不合格标准 |
|--------|------|-----------|
| **弯曲度** | 放在平整台面上，用塞尺测缝隙 | 中间拱起 > 2mm → 内部可能有分层 |
| **两侧铝壳平整** | 手摸+目测 | 凹凸、划痕深及壳体 → 可能影响散热 |
| **极柱方向确认** | 万用表正反测量确认极性 | 极柱松动 → 内部焊接断裂 |
| **长度一致性** | 卷尺量 | 同批次 ±1mm 以内，差异太大 → 不同厂/不同批次拼凑 |
| **壳体温度** | 放电 10 分钟后手摸 | 局部热点 → 内部短路/极片不均匀 |

### 8.3 刀片电池内阻特殊说明

```
刀片电池因为极片长（96cm），本身内阻比同容量的方壳略高一点：

例：
  100Ah 方壳磷酸铁锂：内阻 ~0.27mΩ
  138Ah 刀片电池：    内阻 ~0.20mΩ（容量大了，内阻低了）

  但换算到单位容量：
  100Ah 方壳：0.27 × 100 = 27  （参考值）
  138Ah 刀片：0.20 × 138 = 27.6（参考值）

  → 刀片的内阻/容量比与方壳基本一致
  → 判断标准可以直接用上面磷酸铁锂的表格
```

## 9. 组装成组的额外检查项

如果买的是一整组分好容的电池组（如 16 串 48V 100Ah），还要检查：

### 9.1 收到货必做检查

```
1. 万用表测整组总电压 → 应在标称电压 ±10% 内
   例如 16S LFP 标称 51.2V，应在 46~58V 之间

2. 万用表逐串测电压 → 每串电压差应 < 0.05V
   如果某串明显偏低（如 3.05V 而其他 3.25V）
   → 该串自放电大或有内短

3. 外观检查保护板（BMS）
   - 连接线是否牢靠
   - 是否有烧灼痕迹
   - 排线插头有无松脱
```

### 9.2 带保护板放电测试

```
1. 接上负载（如逆变器 + 电热器）
2. 用库仑计串联，记录放出的总电量
3. 观察 BMS 各串电压显示（APP/上位机）
4. 重点看：
   - 某串在放电过程中电压下降最快 → 该串容量最低
   - 这就是整组电池的短板
```

## 10. 验收判定标准速查表

### 综合判定——买回来的二手电池值不值得留

| 维度 | ✅ 可留 | ⚠️ 可留但压价 | ❌ 退货 |
|------|--------|-------------|--------|
| **外观** | 无鼓包/漏液/变形 | 轻微划痕 | 鼓包/漏液/凹坑 |
| **电压** | 在正常区间 | 稍偏低但可充回 | 0V 或 > 满电电压 |
| **内阻** | 不超过新电池 150% | 150%~200% | > 200% 新增内阻 |
| **容量** | ≥ 85% 标称 | 70%~85% | < 70% |
| **自放电** | 72h 压降 < 0.03V | 0.03~0.08V | > 0.08V |
| **一致性**（同批） | 内阻波动 < ±15% | ±15%~±30% | > ±30% |
| **生产日期** | < 3 年 | 3~5 年 | > 5 年 |

### 快速验收流程（收到货 30 分钟内做完）

```
Step 1: 拆包，所有电池在平整台面上排列（2 分钟）
Step 2: 目视外观，排除鼓包漏液（3 分钟）
Step 3: 万用表逐块测电压，记录（5 分钟）
Step 4: 内阻仪逐块测内阻，记录（10 分钟）
Step 5: 排除电压异常和内阻超标的电池（5 分钟）
Step 6: 随机抽 1~2 块上放电仪测容量（需要几个小时，但这一步不用等）

→ 如果前 5 步都过了，基本问题不大
→ 第 6 步是最终验收，如果有问题再找卖家
```
