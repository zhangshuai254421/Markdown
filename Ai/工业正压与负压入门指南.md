## 1. 什么是压力——先搞懂这个概念

### 1.1 最直观的理解

```
力 = 推/拉的劲
压力 = 这个劲平均分摊到每块面积上有多少

P = F ÷ S
压力 = 力 ÷ 面积

同样 100kg 的力：
  - 压在针尖上（面积极小）→ 压强巨大 → 针可以扎穿东西
  - 压在桌面上（面积很大）→ 压强很小 → 桌面没啥感觉
```

### 1.2 压力的单位

| 单位 | 说明 | 什么时候用 |
|------|------|-----------|
| **Pa（帕斯卡）** | 国际标准单位，1N 的力压在 1m² 上 | 科学计算 |
| **kPa（千帕）** | 1000 Pa | 工业常见 |
| **MPa（兆帕）** | 1,000,000 Pa | 液压、气压系统 |
| **bar（巴）** | 约等于 1 个大气压 | 欧洲常见 |
| **atm（标准大气压）** | 1 atm = 101.325 kPa | 海平面大气压 |
| **kgf/cm²** | 公斤力每平方厘米 | 国内工厂老设备常见 |
| **mmHg（毫米汞柱）** | 多少毫米高度的水银柱 | 医疗血压计 |
| **mmH₂O（毫米水柱）** | 多少毫米高度的水柱 | 微小压力/负压 |

### 1.3 换算记住几个关键的就行

```
1 个标准大气压 (atm) = 101.325 kPa
                      ≈ 1.013 bar
                      ≈ 1.033 kgf/cm²
                      ≈ 760 mmHg
                      ≈ 10.33 米水柱

日常记忆：
  1 bar ≈ 1 公斤压力 ≈ 1 个大气压 ≈ 100 kPa
```

## 2. 大气压——你看不见但无时无刻不在的东西

### 2.1 大气压是什么

```
地球表面有一层厚厚的大气层（约 100km 厚）。

空气虽然很轻，但 100km 厚的空气柱压下来，
每平方米大约有 10 吨的重量。

你头顶上这个 10 吨/m² 的压力 = 1 个大气压

为什么你没被压扁？
→ 因为你的身体内部也有相等的压力向外推（内外平衡）
→ 就像在水里时，你会浮起来而不是被压到水底
```

### 2.2 大气压会变

```
海拔越高 → 空气柱越短 → 气压越低

  - 海平面：101.3 kPa（标准）
  - 拉萨（3650m）：约 65 kPa（只有海平面的 64%）
  - 飞机巡航高度（10000m）：约 26 kPa（只有海平面的 26%）
  - 珠峰顶：约 33 kPa

高原上烧水为什么不到 100°C？
→ 气压低 → 水的沸点降低 → 80°C 水就沸腾了
```

### 2.3 这两个实验让你彻底理解大气压

```
实验 1：马德堡半球实验（1654年，经典）

  两个铜制半球扣在一起，把里面抽成真空。
  结果：16 匹马都拉不开！

  原因：球内没有气压向外推，球外的大气压把两个半球死死压在一起。
        大气压力 ≈ 10 吨/m² × 球的截面积 → 力非常大

实验 2：吸管喝水（每天都在做）

  你用嘴吸吸管 → 吸管内的气压降低了
  → 吸管内的气压 < 外面的大气压
  → 大气压把液体"推"进吸管，推到你嘴里
  → 不是你把水"吸"上来的，是大气压把水"压"上来的！
```

## 3. 伯努利原理与文丘里效应——负压的物理基础

### 3.1 伯努利方程——一句话看懂

```
流体（气体/液体）沿管道流动时，能量在两种形式之间互相转化：

  静压力（P）  +  动压力（½ρv²）  +  位能（ρgh）  =  常数

  P      = 流体对管壁的压力（压力表测的就是这个）
  ½ρv²   = 流体运动的动能（速度越快，这项越大）
  ρgh    = 高度差产生的势能（水平管道可以忽略）
```

**核心规律**：

```
总能量不变的前提下：

  截面大 → 流速慢 → 动压小 → 静压大（高压区）
  截面小 → 流速快 → 动压大 → 静压小（低压区，可低到负压！）
```

### 3.2 管道截面变化 → 压力变化（图解）

```
水平管道，从左到右：

  ═══════════╦══════════╗═══════════════
     宽截面   ║  窄截面   ║    宽截面
    （低速）   ║ （高速）   ║   （低速）
              ║          ║
  静压 = 高   ║ 静压 = 低 ║  静压 = 高
  动压 = 低   ║ 动压 = 高 ║  动压 = 低
              ║          ║
           ┌──╨──┐       
           │ 负压 │  ← 如果窄到足够程度，这里的静压
           │ 区域 │     会降到低于大气压 → 负压！
           └─────┘      
```

### 3.3 日常中的伯努利效应

```
场景 1：两张纸之间吹气
  拿两张纸，中间留一点缝隙。
  往缝隙里吹气 → 纸不但不分开，反而靠拢！

  原因：纸之间的气流速度快 → 静压低 → 纸外侧的大气压把纸压向中间

场景 2：地铁/高铁站台黄线
  列车高速进站 → 列车与站台之间的空气被高速带动
  → 该区域的静压力骤降 → 人可能被大气压"推"向列车方向
  → 所以要画黄线，让你站在安全距离外

场景 3：喷雾瓶/香水瓶
  捏气囊 → 一股气流高速通过竖直吸管的上方
  → 吸管上方气流快、压力低（负压）
  → 瓶内液体被大气压推上吸管 → 被气流吹散成雾

场景 4：飞机机翼升力
  机翼上表面弧度大 → 空气流过时路程长、速度快
  机翼下表面平直 → 空气流过时速度慢
  → 上表面压力低、下表面压力高 → 产生升力
```

### 3.4 文丘里管——伯努利原理的工程应用

**文丘里管（Venturi Tube）** 是伯努利原理最直接的工程应用。它的结构刻意制造截面变化来实现压力控制：

```
        ┌──────────────────────────────────┐
        │      文丘里管三要素               │
        │                                  │
  入口  │  收缩段      喉部      扩散段     │  出口
  ────▶ │  ╲        ┌───┐       ╱         │ ────▶
  大截面│   ╲       │   │      ╱          │ 大截面
  低速  │    ╲      │ ● │     ╱           │ 低速
  高压  │     ╲     │   │    ╱            │ 压力恢复
        │      ╲    └───┘   ╱             │
        │       ╲    ↑    ╱              │
        │        ╲  低压  ╱               │
        │         ╲ 区域 ╱                │
        └──────────────────────────────────┘
                   ↑
              在这个位置开孔
              → 负压取气口
```

| 部位 | 截面变化 | 速度变化 | 压力变化 | 作用 |
|------|:---:|:---:|:---:|------|
| **收缩段** | 大→小 | 慢→快 | 高→低 | 加速气流，把压力能转为动能 |
| **喉部** | 最小 | 最快 | 最低（负压） | 产生负压的取气点 |
| **扩散段** | 小→大 | 快→慢 | 低→高 | 减速气流，动能转回压力能 |

### 3.5 文丘里管 vs 孔板——别搞混了

```
文丘里管（Venturi Tube）：
  - 平滑渐缩 + 平滑渐扩
  - 压力损失小（≈10~20%）
  - 用于需要回收能量的场合

孔板（Orifice Plate）：
  - 一块板中间打孔，突变截面
  - 压力损失大（≈50~80%）
  - 用于差压流量测量（不在乎能量损失）

共同点：都利用截面变化产生压力差。
区别：文丘里讲究"平滑过渡，减小损失"；孔板不管损失，只要差压信号。
```

## 4. 正压和负压的定义

### 3.1 以大气压为参照物

```
工业上说"正压"还是"负压"，都是在跟大气压比较：

  正压（Positive Pressure） = 比大气压高
  负压（Negative Pressure） = 比大气压低
  零压                      = 等于大气压（跟外面一样）
```

### 3.2 用数字说话

```
以大气压为 0 作为参照：

  +100 kPa = 比大气压高 100 kPa（正压）
  +50 kPa  = 比大气压高 50 kPa （正压）
  0 kPa    = 等于大气压（零压）
  -10 kPa  = 比大气压低 10 kPa（负压）
  -50 kPa  = 比大气压低 50 kPa（负压）
```

### 3.3 一句话总结

```
正压 = 容器/管道里的气，想往外跑（内部压力 > 外部）
负压 = 容器/管道里的气，外面的想往里进（内部压力 < 外部）

正压 → 气往外漏
负压 → 气往里吸

正压 → 鼓风机/气泵往里吹气
负压 → 抽气机/真空泵往外抽气
```

## 5. 用大白话理解正压和负压

### 4.1 生活中常见的例子

```
正压（比你想象的多得多）：
  - 气球吹鼓了 → 内部正压（比外面大）
  - 自行车轮胎打满气 → 胎内正压（约 300~500 kPa，即 3~5 个大气压）
  - 高压锅盖上后 → 锅内正压（约 80 kPa，水可以到 120°C 才沸腾）
  - 打开可乐罐"嘶"一声 → 罐内正压释放
  - 长江三峡大坝底部 → 水压巨大（也是正压！液体压力也一样）
  - 消防水带喷水 → 正压驱动

负压（你每天在用但没意识到）：
  - 吸管喝饮料 → 你嘴里是负压
  - 吸尘器 → 内部是负压，把灰尘吸进去
  - 真空包装袋 → 袋内负压，袋子紧贴在食物上
  - 马桶水箱注水 → 虹吸作用，负压
  - 打针/抽血 → 针管拉出时内部是负压
  - 吸盘挂钩 → 吸盘内部是负压，外部大气压把它"压"在墙上
```

### 4.2 两句话一辈子忘不了

```
正压 = 想"吐" → 里面的东西想往外跑
负压 = 想"吸" → 外面的东西想往里进

你去工厂看到一个管道：
  - 如果漏气了往外喷 → 正压管道
  - 如果漏气了往里吸 → 负压管道
```

## 6. 正压和负压是怎么产生的

### 5.1 产生正压——把东西往里送

```
常见正压源：

  1. 空气压缩机 / 空压机
     → 把空气压缩进储气罐 → 罐内正压
     → 工厂里最常见的正压源

  2. 鼓风机 / 风机
     → 持续往管道/容器里吹气
     → 用于通风、燃烧供氧等

  3. 泵（水泵/油泵/液压泵）
     → 把液体加压输送到管道里
     → 所有液压系统、供水系统

  4. 加热（封闭空间）
     → 封闭容器加热 → 气体膨胀 → 压力升高
     → 高压锅就是这个原理
```

### 5.2 产生负压——把东西往外抽

```
常见负压源：

  1. 真空泵
     → 把容器里的气体抽出去
     → 从"微负压"到"接近真空"都行

  2. 引风机
     → 在管道末端抽气，让管道内产生负压
     → 工厂排烟、除尘

  3. 文丘里管（Venturi）
     → 气流通过狭窄处，速度加快，压力降低
     → 没有活动件，纯靠物理原理产生负压

  4. 冷凝（封闭空间）
     → 封闭容器内的蒸汽冷却变回水 → 体积坍缩 → 负压
     → 没放气的蒸汽管道停机后内部会形成负压，可能把管道吸瘪！
```

## 7. 工业中最常见的正压场景

### 6.1 压缩空气系统

```
场景：几乎每个工厂都有

空压机 → 储气罐 → 干燥机 → 主管道 → 各用气点

压力值：0.4 ~ 1.0 MPa（约 4~10 个大气压）

用途：
  - 气动工具（气扳手、气钻）
  - 气动阀门（你的设备里可能就有）
  - 吹扫清洁
  - 仪表风气源
```

### 6.2 锅炉与蒸汽管道

```
压力值：
  低压锅炉：< 1.6 MPa (16 公斤)
  中压锅炉：1.6 ~ 3.8 MPa
  高压锅炉：> 3.8 MPa (火力发电厂)

管道里的都是正压——漏了会往外喷高温蒸汽！
```

### 6.3 液压站与液压油管

```
压力值：
  一般液压系统：10 ~ 21 MPa
  工程机械：20 ~ 35 MPa
  超高压：> 35 MPa

液压油在管道里是正压 → 油管爆了能从针孔大小的漏点射出，
穿透皮肤会致死（液压油注射伤）！
```

### 6.4 正压送风系统

```
大型建筑、车间、地下室的送风：
  - 鼓风机把室外新鲜空气加压送进来
  - 室内变成微正压（50~200 Pa）
  - 目的：防止脏空气/灰尘从门缝倒灌进来
```

### 6.5 工艺设备正压运行

```
反应釜、蒸馏塔、换热器等：
  - 根据工艺需要保持正压运行
  - 防止空气倒灌 → 防止氧化、防止形成爆炸混合气
```

## 8. 工业中最常见的负压场景

### 7.1 真空系统（负压的极端表现）

```
真空度等级（从微负压到高真空）：

  粗真空    ：100 kPa ~ 100 Pa
  中真空    ：100 Pa ~ 0.1 Pa
  高真空    ：0.1 Pa ~ 10⁻⁵ Pa
  超高真空  ：< 10⁻⁵ Pa

工业常见：粗真空 ~ 中真空
```

| 真空度 | 绝对压力 | 常见用途 |
|--------|:------:|---------|
| 轻微负压 | 80~100 kPa | 除尘、通风 |
| 低真空 | 10~80 kPa | 真空吸盘搬运、真空包装 |
| 中真空 | 0.1~10 kPa | 真空镀膜、冷冻干燥、真空注油 |
| 高真空 | < 0.1 Pa | 电子束焊、半导体工艺 |

### 7.2 除尘系统（负压用得最多的地方）

```
工厂除尘原理：

  吸尘罩（负压）→ 管道（负压）→ 除尘器 → 风机（抽气）→ 排空

整个管道都是负压的：
  - 含尘气体被"吸"进管道
  - 管道漏了 → 空气往里吸 → 不会冒外面的灰
  - 风机在系统的最末端往外抽

跟你家的吸尘器一个原理，只是大几千倍。
```

### 7.3 真空吸盘搬运

```
玻璃厂、金属板材厂、电子厂常见：

  真空泵 → 吸盘内部抽成负压
  → 外部大气压把工件"压"在吸盘上
  → 工件被提起来
  → 释放负压 → 工件放下

一个 10cm×10cm 的吸盘在大气压下能吸住约 100kg
（不用粘不用胶，纯靠气压差）
```

### 7.4 真空注油/真空浸漆

```
场景：变压器/电机线圈浸漆

  把工件放进真空罐 → 抽负压 → 线圈里的空气被抽出来
  → 注入绝缘漆 → 漆液被大气压"推"进线圈缝隙
  → 比直接泡进去强得多，因为有负压 → 气泡先排掉了
```

### 7.5 引风排烟系统

```
锅炉/窑炉/干燥线的尾部：

  烟囱或引风机在尾部抽 → 炉膛/管道内形成负压
  → 燃烧产生的烟气被"吸"走，不会从炉门冒出来
  → 负压太大 → 外面空气被吸入，降低炉温
  → 负压太小 → 炉门冒烟，车间里全是烟
```

### 7.6 真空干燥

```
场景：食品/药品/化工原料干燥

  物料放在密闭罐里 → 抽真空（负压）
  → 水的沸点降低（负压下 60°C 甚至 40°C 水就能沸腾蒸发）
  → 低温干燥 → 不破坏热敏成分
  → 比热风干快而且不烤焦

冻干咖啡/冻干水果片就是这么做的。
```

## 9. 表压、绝压、差压——三种压力测量方式

### 8.1 一张图彻底搞清楚

```
                    压力值
                      ↑
       +200 kPa ──    │    ← 表压 200 kPa（比大气压高 200 kPa）
                      │
       +100 kPa ──    │    ← 表压 100 kPa
                      │
  ───────── 0 ────  大气压（101.325 kPa 绝压）← 表压的零点
                      │
       -50 kPa  ──    │    ← 表压 -50 kPa（比大气压低 50 kPa）
                      │
  ────────────────  0 绝压（完全真空）

  表压 = 绝压 - 大气压
  绝压 = 表压 + 大气压
```

### 8.2 什么时候用什么

| 测量方式 | 零点参照 | 什么时候用 |
|----------|---------|-----------|
| **表压（Gauge）** | 大气压 = 0 | 绝大多数场合（气动/液压/水管/锅炉） |
| **绝压（Absolute）** | 完全真空 = 0 | 需要精确知道气体量（如真空工艺、气象） |
| **差压（Differential）** | 高压侧 - 低压侧 | 滤芯堵塞检测、流量测量、洁净室 |

### 8.3 差压——工业巡检必懂

```
差压 = 两个位置的压力差值

最常见用途：

1. 过滤器前后差压：
     干净过滤器：差压 = 0~10 kPa
     过滤器堵了一半：差压 = 20~50 kPa
     过滤器快堵死了：差压 = 50~100 kPa
     → 看到差压升高就知道该换滤芯了

2. 孔板流量计：
     管道里装一个带孔的板 → 孔板前后有压差
     → 差压大 = 流量大 → 通过差压算流量

3. 洁净室：
     正压车间内外差压 = 车间内部 - 外部 ≈ 15~50 Pa
     → 差压不够 → 脏空气可能倒灌
```

## 9. 正压和负压怎么测量

### 9.1 测压仪表全家福

```
┌─────────────────────────────────────────────┐
│            压力测量仪表                        │
├─────────────┬──────────────┬────────────────┤
│   机械式    │   电子式      │   开关式        │
├─────────────┼──────────────┼────────────────┤
│ 弹簧管压力表│ 压力变送器    │ 压力开关        │
│ 膜片压力表  │ 差压变送器    │ 压差开关        │
│ U 形管      │ 数显压力表    │                │
│ 膜盒压力表  │              │                │
└─────────────┴──────────────┴────────────────┘
```

### 9.2 常见仪表的读数

#### 弹簧管压力表（波登管/Bourdon 管）

```
最常见、最便宜的压力表（每个工厂都有几十个）

  一个圆形表盘，中间一根指针：
  
    指针在 0 → 管道压力 = 大气压
    指针在 0.5  → 比大气压高 0.5 MPa（约 5 公斤）
    指针在负值  → 负压！

  ⚠️ 负压表的 0 和正压表的 0 位置不一样！
    正压表：0 在表盘最左边（不能显示负压）
    真空表：0 在表盘右边，往左转是负压
    复合表：0 在中间，正压往右，负压往左
```

#### 压力变送器（4-20mA 信号）

```
把你设备上读到的模拟量换算成压力：

  4mA  = 测量下限（如 0 kPa 或 -100 kPa）
  20mA = 测量上限（如 1000 kPa）
  
  读到的电流值 → PLC/DCS 换算出压力值

例：
  传感器量程 -100 ~ +900 kPa
  当前电流 = 12mA

  比例 = (12-4) / (20-4) = 0.5
  压力 = -100 + 0.5 × (900-(-100)) = -100 + 500 = 400 kPa（正压）
```

#### U 形管压力计

```
最简单、最准确、但只能人工读数的测压方法：

一根 U 形透明管，里面装水或水银，一端接被测点，一端通大气

  两边液面一样高 → 被测压力 = 大气压
  被测端液面低，另一端液面高 → 正压，差值就是压力
  被测端液面高，另一端液面低 → 负压，差值就是负压值

  正压 100mm 水柱 = 约 1 kPa（很小的压力）
  这就是为什么洁净室都用 Pa 为单位（1 Pa ≈ 0.1mm 水柱）
```

## 10. 正压房与负压房

### 10.1 概念对比

```
正压房（Positive Pressure Room）：
  里面气压 > 外面气压
  空气从里往外流
  目的：保护房间里的东西不被外面污染

  例子：
    - 芯片制造洁净车间（ISO 1~9 级）
    - 手术室
    - 食品/药品灌装车间
    - 数据中心（防止灰尘）

负压房（Negative Pressure Room）：
  里面气压 < 外面气压
  空气从外往里流
  目的：保护外面的东西不被里面的污染

  例子：
    - 医院隔离病房（传染病人的病菌不飘出去）
    - 生物安全实验室（P3/P4 级）
    - 化工厂有毒有害区域
    - 喷漆房（漆雾不飘到外面车间）
    - 打磨/抛光车间（粉尘不外泄）
```

### 10.2 正负压房的实现原理

```
正压房的实现：
  送风量 > 排风量 → 多出来的空气从门缝、泄漏口往外跑 → 正压
  压差控制：通常 +15~50 Pa（相对走廊/室外）
  洁净等级越高，正压越大

负压房的实现：
  排风量 > 送风量 → 外面的空气从门缝、泄漏口往里补 → 负压
  压差控制：通常 -15~-50 Pa（相对走廊/室外）
  排出去的气体要经过处理（过滤/消毒/排放）
```

| 设备 | 正压/负压 | 功能 |
|------|:------:|------|
| **空压机** | 产生正压 | 把空气压缩成高压气源 |
| **真空泵** | 产生负压 | 把密闭空间的气体抽走 |
| **鼓风机** | 产生正压 | 大流量送风 |
| **引风机** | 产生负压 | 大流量抽风/排烟 |

| **安全阀** | 释放正压 | 压力超过设定值时自动打开泄压 |
| **止回阀** | 防止回流 | 只允许单向流动 |
| **真空吸盘** | 利用负压 | 搬运工件 |

| **压力表** | 显示正/负压 | 人工读数 |

## 12. 安全注意事项

### 12.1 正压系统——防爆防喷

```
🚫 正压管道带压拆卸 → 高压气/液喷出伤人
🚫 压力表损坏/过期不换 → 无法知道真实压力 → 超压爆炸
🚫 安全阀卡死/被堵 → 超压无保护 → 管道/容器炸裂
🚫 液压油管破损 → 高压油注射进皮肤 → 必须立即手术

✅ 检修正压管道前 → 先泄压！
✅ 安全阀定期校验 → 一般一年一次
✅ 压力表定期检定 → 超期更换
✅ 储气罐定期排水 → 防止锈蚀和积水占容积
```

### 12.2 负压系统——防吸瘪防倒灌

```
🚫 负压管道抽过头 → 外部大气压太大 → 薄壁管道/容器被"压瘪"
    （真空度过高对容器来说反而是"外压"巨大）
🚫 真空系统突然漏气 → 空气高速冲入 → 可能带进杂质/水分/氧气
🚫 负压排水 → 可能虹吸倒灌，把污水抽回清水管道

✅ 负压容器要做外部加强筋 → 防止瘪掉
✅ 真空系统要有破真空阀 → 需要时打开恢复常压
✅ 排水管道装止回阀 → 防止虹吸倒灌
```

### 12.3 正压负压切换的管道——特别注意

```
场景：蒸汽管道停机冷却后

  运行时：蒸汽正压
  停机冷却后：蒸汽冷凝成水 → 体积坍缩 → 管道变成负压

  如果管道上没有真空破除阀：
    - 外部大气压可能把管道/换热器瘪进去
    - 如果管道有漏点会吸入空气 → 下次启动时管道里有空气
      → 可能产生水锤现象 → 管道爆裂
```

## 附：一句话复习

```
正压：里面比外面高 → 东西想往外跑 → 吹/压/推
负压：里面比外面低 → 东西想往里跑 → 吸/抽/拉

读懂压力表：
  指针在 0 右边 → 正压
  指针在 0 左边 → 负压

工厂安全第一句话：
  检修先泄压，真空要破空。
```
