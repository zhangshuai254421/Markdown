## 1. 为什么要自制微距镜头

一支原厂微距镜头（如佳能百微、尼康 105mm Micro）动辄四五千到上万元。对于刚入门微距摄影的爱好者，自制方案能以 1/10 甚至更低的成本体验微距拍摄：

| 方案 | 成本（元） | 放大倍率 | 画质 | 难度 |
|------|:--------:|:-------:|:----:|:----:|
| 反接镜头 | 20 ~ 100 | 1:1 ~ 3:1 | ★★★★ | ★☆☆ |
| 近摄接圈 | 30 ~ 200 | 0.5:1 ~ 2:1 | ★★★★ | ★☆☆ |
| 近摄镜片 | 50 ~ 500 | 0.3:1 ~ 1:1 | ★★★ | ★☆☆ |
| 镜头堆叠 | 30 ~ 150 | 2:1 ~ 10:1 | ★★★★ | ★★☆ |
| 皮腔延伸 | 100 ~ 500 | 1:1 ~ 5:1 | ★★★★★ | ★★★ |

## 2. 基础光学原理

了解几个关键概念，有助于理解每种方案的原理：

### 2.1 放大倍率

```
放大倍率 = 传感器上成像尺寸 ÷ 被摄物体实际尺寸

例：一只 10mm 长的蚂蚁，在传感器上成像也是 10mm → 1:1（1 倍）
```

- **1:1**（1×）：真正的微距门槛，物体与成像等大
- **2:1**（2×）：成像比物体大一倍，进入超微距领域
- **1:2**（0.5×）：近摄，还不是严格意义的微距

### 2.2 像距与放大倍率的关系

```
                             镜头
    被摄体 ○═══════════════|--------|═══════════════○ 传感器
             物距                     像距

    放大倍率 M = 像距 ÷ 物距
    镜头公式：1/f = 1/u + 1/v
    （f = 焦距，u = 物距，v = 像距）
```

> **核心结论**：要让放大倍率变大，要么**缩短物距**（靠近被摄体），要么**拉长像距**（镜头离传感器更远）。所有 DIY 方案本质上都是在做这两件事。

### 2.3 法兰距

法兰距（Flange Focal Distance）是镜头卡口到传感器的距离。各家不同：

| 卡口 | 法兰距 |
|------|:------:|
| 佳能 EF | 44.0mm |
| 尼康 F | 46.5mm |
| 索尼 E | 18.0mm |
| 富士 X | 17.7mm |
| M4/3 | 19.25mm |
| 宾得 PK | 45.46mm |

> 法兰距的差异决定了反接和转接方案的可行性——无反相机（短法兰距）是天生的转接神器。

## 3. 方案一：反接镜头

### 3.1 原理

将镜头**反过来**安装，使原本朝向传感器的后镜组朝外、原本朝外的前镜组朝向传感器。这极大改变了光学结构，使普通镜头变成一支高倍微距镜头。

```
正常安装：     [被摄体] → → → [前镜组]═══[后镜组] → → [传感器]

反接安装：     [被摄体] → → [后镜组]═══[前镜组] → → → [传感器]
                            ↑ 此时后镜组面朝被摄体
```

### 3.2 适用镜头

- **最佳选择**：50mm 以下定焦镜头（28mm、35mm、50mm）
- **可尝试**：标准变焦镜头的广角端
- **不适合**：长焦镜头（反接后放大倍率反而变小）

> 💡 **镜头焦距越短，反接后放大倍率越大。**
> - 50mm 反接 ≈ 1:1
> - 28mm 反接 ≈ 2:1 ~ 3:1
> - 24mm 反接 ≈ 3:1 ~ 4:1

### 3.3 所需材料

| 材料 | 用途 | 价格 |
|------|------|:----:|
| 反接环（ reversing ring） | 连接机身与倒装镜头 | 15 ~ 50 元 |
| 老镜头（50mm f/1.8 等） | 被反接的镜头 | 100 ~ 300 元 |
| **或者**：双阳环 | 反向连接两个镜头 | 10 ~ 30 元 |

### 3.4 操作步骤

**方式 A：反接环直接上机身**

```
[机身] ← [反接环] ← [镜头（倒装）]
```

1. 购买对应卡口的反接环（例如佳能 EF 口就买 EF 反接环）
2. 将反接环拧在**机身卡口**上
3. 将镜头**反方向**拧在反接环的滤镜螺纹上（反接环前端有滤镜螺纹）
4. 确认拧紧，完成

```
┌──────────┐      ┌──────────┐      ┌─────────────────┐
│   机身    │ ←→  │  反接环   │ ←→  │ 镜头（滤镜螺纹） │
│ EF 卡口   │      │ 卡口+螺纹 │      │ 前后反过来了      │
└──────────┘      └──────────┘      └─────────────────┘
```

**方式 B：双阳环镜头对扣（详见第 6 章）**

### 3.5 反接后的变化

| 功能 | 状态 |
|------|:----:|
| 自动对焦 | ❌ 不可用 |
| 光圈控制 | ⚠️ 视镜头而定 |
| 防抖 | ❌ 不可用 |
| EXIF 信息 | ❌ 不可用 |
| 测光 | ⚠️ 需手动补偿 |

### 3.6 光圈控制技巧

不同镜头反接后的光圈状态：

```
电子光圈镜头（如佳能 EF 50mm f/1.8 STM）：
   → 反接后光圈锁定在最小光圈（无法拍照）
   → 解决方法：先正常安装在机身上，设好光圈，按住景深预览键的同时
     拆下镜头 → 光圈卡在设定值 → 再反接安装

手动光圈环镜头（如尼康 AI-S、老手动头）：
   → 反接后可以直接拧光圈环 ✅（强烈推荐用这种）
```

> ⚠️ **强烈建议使用带手动光圈环的老镜头做反接**，省去光圈控制的麻烦。百元左右的 50mm f/1.8 手动头是最佳入门选择。

### 3.7 反接镜头的保护措施

反接后后镜组裸露在外，容易进灰或受损：

```
方案 1：用 UV 滤镜反装（滤镜螺纹拧在反接环上）保护后镜组
方案 2：用镜头后盖开孔，扣在镜头上
方案 3：3D 打印一个后镜组保护罩
方案 4：平时不拍就拆下来，镜头前后盖都盖好
```

## 4. 方案二：近摄接圈

### 4.1 原理

在机身和镜头之间加入延长筒，增加像距，使镜头能对焦更近的物体。

```
正常：   [镜头] ←──────── 44mm ────────→ [机身]
                           法兰距

接圈：   [镜头] ←────── 44mm + 接圈长度 ──────→ [机身]
                          ↑ 像距变大 → 放大倍率变大
```

### 4.2 放大倍率计算

```
原始倍率（以镜头最近对焦距离为准） + 接圈长度 ÷ 镜头焦距 ≈ 新增倍率

例：50mm 镜头，最近对焦距离时放大 0.15×（1:6.7）
   加入 50mm 的接圈 → 新增 50÷50 = 1.0×
   总放大倍率 ≈ 0.15 + 1.0 = 1.15×（超过 1:1）
```

### 4.3 接圈类型

| 类型 | 优点 | 缺点 | 价格 |
|------|------|------|:----:|
| **无触点接圈** | 便宜、简单 | 不能调光圈，手动对焦 | 20 ~ 80 元 |
| **有触点接圈** | 支持自动对焦、光圈控制 | 稍贵 | 100 ~ 300 元 |
| **可调长度接圈（皮腔式）** | 灵活变倍 | 体积大 | 150 ~ 400 元 |

> 💡 推荐入门买**有触点的成套接圈**（通常含 12mm + 20mm + 36mm 三节），可任意组合出多种长度。

### 4.4 操作步骤

1. 取下镜头
2. 将接圈安装在**机身卡口**上
3. 将镜头安装在**接圈前端**
4. 多节接圈可以自由组合（12+20+36=68mm 等）
5. 开始拍摄

```
[机身] ← [20mm接圈] ← [36mm接圈] ← [镜头]
```

### 4.5 注意事项

- 接圈越长，**进光量越少**，需要补光或提高 ISO
- 接圈越长，**景深越浅**，对焦越困难
- 部分超长接圈组合可能导致镜头无限远不合焦（正常现象，微距也不拍无限远）
- 电子触点接圈要买**匹配自己卡口**的版本

## 5. 方案三：近摄镜片

### 5.1 原理

在镜头前加装一片凸透镜（类似放大镜），缩短镜头的最近对焦距离。

```
正常：   镜头最近对焦 30cm

加近摄镜：镜头可以凑到 10cm 甚至更近 → 放大倍率变大
```

这是**最简单、最方便**的方案，直接拧在镜头滤镜螺纹上即可。

### 5.2 近摄镜的屈光度

近摄镜的强度用屈光度（Diopter）表示，数字越大放大能力越强：

```
+1  → 放大能力弱，适合花草特写
+2  → 中等放大
+4  → 较强放大
+8  → 强放大（但画质下降明显）
+10 → 极强放大（多为消色差双合镜片）
```

### 5.3 近摄镜类型

| 类型 | 结构 | 画质 | 价格 |
|------|------|:----:|:----:|
| 单片近摄镜 | 单层玻璃 | ★★☆☆☆ | 20 ~ 80 元 |
| 消色差近摄镜（APO） | 双合镜片 | ★★★★☆ | 200 ~ 600 元 |

> ⚠️ **单片近摄镜会产生明显的色差和边缘模糊**，适合体验，不适合严肃创作。如果预算允许，直接上消色差近摄镜（如佳能 500D 近摄镜、雷克沙 DCR-250、尼康 5T/6T 等）。

### 5.4 操作步骤

1. 确认镜头滤镜口径（如 52mm、58mm、67mm、72mm、77mm）
2. 购买对应口径的近摄镜，或买大口径 + 转接环适配多支镜头
3. 将近摄镜拧在镜头前方滤镜螺纹上
4. 自动对焦和测光**正常工作**（这是相比反接的最大优势）
5. 开始拍摄

### 5.5 叠加使用

近摄镜可以叠加（+2 和 +4 一起 ≈ +6），但画质会进一步下降：

```
镜头 ← [+4近摄镜] ← [+2近摄镜]
              ↑ 高屈光度靠近镜头安装，画质更好
```

## 6. 方案四：镜头堆叠

### 6.1 原理

将两支镜头通过双阳环（也称"对扣环""耦合环"）连接，一支正常安装在机身上（称为"主机镜头"），另一支反接在前方（称为"反接镜头"）。

```
[机身] ← [主机镜头（正常安装）] ← [双阳环] ← [反接镜头（倒装）]
  ↑          ↑ 例如 70-200mm         ↑           ↑ 例如 50mm
机身        长焦镜头              滤镜螺纹×2    定焦镜头
```

放大倍率 = 主机镜头焦距 ÷ 反接镜头焦距

```
200mm（主机）÷ 50mm（反接）= 4× 放大
100mm（主机）÷ 28mm（反接）= 3.6× 放大
```

### 6.2 镜头选择

| 角色 | 推荐 | 原因 |
|------|------|------|
| **主机镜头** | 70-200mm、100mm、135mm 等长焦 | 焦距越长放大倍率越大；变焦可灵活调整 |
| **反接镜头** | 28mm、35mm、50mm 定焦 | 焦距越短放大倍率越大；手动光圈更方便 |

### 6.3 所需材料

| 材料 | 用途 |
|------|------|
| 双阳环（对扣环） | 连接两支镜头的滤镜螺纹 |
| 主机镜头（中长焦） | 正常安装在机身上 |
| 反接镜头（短焦） | 反接，面朝被摄体 |

双阳环规格由**两支镜头的滤镜口径**决定：

```
主机镜头滤镜口径 72mm，反接镜头滤镜口径 52mm
  → 购买 72mm-52mm 双阳环
```

### 6.4 操作步骤

1. 将主机镜头（如 70-200mm）正常安装在机身上
2. 将双阳环拧在**主机镜头前端的滤镜螺纹**上
3. 将反接镜头（如 50mm）**反向**拧在双阳环另一端
4. 反接镜头的光圈开到最大（或按 3.6 节方法锁定）
5. 主机镜头对焦到**无限远**
6. 通过移动机身与被摄体的距离来对焦（不是拧对焦环）

```
操作顺序：
  ① 主机镜头装到机身上
  ② 双阳环拧到主机镜头前端
  ③ 反接镜头反方向拧到双阳环另一端
  ④ 反接镜头光圈全开
  ⑤ 主机镜头 MF，对焦环拧到 ∞
  ⑥ 前后移动相机找焦点
```

### 6.5 画质优化

```
主机镜头设置：
  - 对焦模式：手动对焦（MF）
  - 对焦距离：∞（无限远）
  - 光圈：f/5.6 ~ f/11（最佳光学区间）

反接镜头设置：
  - 光圈：开到最大 f/1.8 ~ f/2.8（避免暗角）
  - 如果可见明显暗角，可略收主机镜头光圈
```

## 7. 方案五：皮腔延伸

### 7.1 原理

皮腔（Bellows）是最灵活的近摄方案。镜头和机身之间由可伸缩的皮腔连接，通过改变长度来调节放大倍率。

```
[机身] ← [皮腔] ← [镜头]
           ↑
     通过旋钮调节伸缩长度
     范围通常 30mm ~ 150mm
```

### 7.2 皮腔类型

| 类型 | 特点 | 价格 |
|------|------|:----:|
| 无触点简易皮腔 | 手动操作，不支持电子控制 | 100 ~ 300 元 |
| 带导轨皮腔 | 有精密对焦导轨，适合微距 | 300 ~ 800 元 |
| 移轴皮腔 | 支持俯仰/摇摆，专业级 | 1000+ 元 |

### 7.3 操作步骤

1. 将皮腔安装在机身上
2. 将镜头安装在皮腔前端
3. 调节皮腔长度到需要的放大倍率
4. 使用导轨或整体移动相机对焦
5. 拍摄

### 7.4 使用技巧

```
- 皮腔 + 放大镜头（如罗德岗 APO-Rodagon）是极致画质方案
- 皮腔 + 普通 50mm 镜头也很实用
- 皮腔越长，曝光补偿越多（可用测光表或试拍确定）
- 务必使用三脚架（皮腔越长震动越明显）
```

## 9. 拍摄技巧与注意事项

### 9.1 对焦技巧

微距拍摄景深极浅（可能不到 1mm），自动对焦基本无用：

```
方法 1：前后移动法（推荐）
  → 设置好放大倍率后，保持对焦环不动
  → 整个人（或相机）前后微移找到焦点
  → 看到最清晰的一瞬间按下快门

方法 2：对焦导轨
  → 相机装在对焦导轨上
  → 拧动导轨旋钮精密前后移动
  → 适合三脚架+精细对焦

方法 3：焦点包围（景深合成）
  → 从近到远连续拍摄多张
  → 每张改变对焦点或前后位置
  → 后期用 Photoshop / Helicon Focus 合成
```

### 9.2 光线与补光

```
问题：微距拍摄（尤其反接/堆叠）工作距离极短（2~10cm），
      镜头/机身本身会挡住光线

解决方案：
  ① 环形闪光灯 / 环形 LED 灯（最佳选择，均匀无影）
  ② 双头微距闪光灯（更灵活的布光）
  ③ 小型 LED 补光灯 + 柔光片
  ④ 闪光灯 + 自制反光板/柔光罩
  ⑤ 自然光 + 反光板（最省钱）
```

### 9.3 防抖与稳定

```
微距三大稳定法则：

1. 三脚架是最有效的防抖工具
   → 选择中轴可横置的三脚架更方便低角度拍摄

2. 使用快门线或 2 秒延时自拍
   → 避免按快门的震动

3. 反光板预升（单反用户）
   → 消除反光板拍击的震动
   → 微单用户无此问题
```

### 9.4 曝光补偿

所有延长像距的方案（接圈、皮腔）都会导致进光量减少：

```
曝光补偿倍数 = (接圈长度 + 镜头焦距)² ÷ 镜头焦距²

例：50mm 镜头 + 50mm 接圈
  = (50+50)² ÷ 50² = 10000 ÷ 2500 = 4 倍
  → 需要 4 倍曝光，即提高 2 档（ISO 或快门速度）
```

> 无反相机可以实时预览画面亮度，比单反方便得多。

### 9.5 工作流程建议

```
1. 装好设备，镜头开到合适光圈
2. 大致设定放大倍率（接圈长度 / 皮腔长度 / 变焦焦距）
3. 将相机架在三脚架上（或手持稳定姿势）
4. 用 LED 灯或自然光照明被摄体
5. 前后微移找到焦点
6. 确认曝光、合焦
7. 用快门线或延时拍摄
8. 回放检查，必要时微调
```

## 10. 常见问题

### Q1：反接后拍出来一片漆黑？

```
原因 ①：电子光圈镜头反接后光圈缩到最小
  → 解决：参考 3.6 节的光圈锁定技巧

原因 ②：快门速度太快 / ISO 太低
  → 解决：降低快门速度或提高 ISO

原因 ③：镜头盖没取下来（别笑，真的发生过）
```

### Q2：反接后拍出来糊成一片？

```
原因 ①：景深太浅（只有 1~2mm）
  → 解决：收光圈到 f/8 ~ f/16

原因 ②：手抖
  → 解决：提高快门速度，或用三脚架

原因 ③：对焦不准
  → 解决：用"前后移动法"对焦（见 9.1 节）
```

### Q3：画面边缘出现暗角（隧道效应）？

```
原因：反接镜头的光圈太小，或镜头堆叠时主机镜头光圈太小
  → 解决：反接镜头光圈全开
  → 镜头堆叠时主机镜头避免收太小光圈
```

### Q4：反接镜头会损坏卡口或电子触点吗？

```
卡口：不会，反接环通过滤镜螺纹连接，不接触卡口
电子触点：没有物理接触，所以也不会有电气损伤
后镜组：暴露在外是唯一风险，注意防尘保护（见 3.7 节）
```

### Q5：无反相机（微单）做微距有什么优势？

```
① 短法兰距（~18mm）— 更容易适配各种转接方案
② 实时取景 — 可以实时看到曝光和放大效果
③ 峰值对焦 — 高亮显示合焦区域，手动对焦更轻松
④ 放大对焦 — 局部放大画面确认焦点
⑤ 无反光板震动 — 天生的微距友好结构
```

### Q6：我需要买什么来开始？

```
最省钱入门组合（<100 元）：
  ① 反接环 ×1（匹配你的卡口）- 15 元
  ② 二手手动 50mm f/1.8 镜头 ×1 - 80 元
  → 总计约 95 元，即可体验 1:1 微距

推荐入门组合（<200 元）：
  ① 有触点近摄接圈套装 ×1 - 150 元
  ② 用你现有的镜头即可
  → 自动功能保留，操作最接近正常拍摄
```
