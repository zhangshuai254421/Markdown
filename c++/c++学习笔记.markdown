### 2.1 输入输出

#### 输出运算符 `<<`

输出运算符 `<<` 用于将数据输出到标准输出流（如 `std::cout`）。以下是一个简单的示例：

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10;
    cout << "The value of a is: " << a << endl;
    return 0;
}
```

在上述代码中，`<<` 将字符串和变量的值依次输出到控制台。
